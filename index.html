Sobre mim

Meu primeiro contato com programação foi durante o ensino médio, onde aprendi a programar microcontroladores no curso técnico usando C. Mais tarde, cursei Engenharia Elétrica por 2 semestres, onde aprendi programação mais a fundo com o mesmo fim, mas dessa vez desenvolvi mais interesse. Saí da Engenharia Elétrica e comecei a cursar o bacharelado em Física. Ali participei do programa de Iniciação Científica por 3 anos desenvolvendo pipelines em Python para análise de dados de galáxias. Nesse período, por pura curiosidade, levei o aprendizado de Python mais a sério.

Considero que uma das melhores formas de aprender uma linguagem de programação são pequenos projetos feitos do zero. Aqui você pode ver alguns dos projetos que me ajudaram a melhorar como programador. Agora, perto do fim da graduação estou buscando minha entrada para o mercado de trabalho e nada melhor do que trabalhar com aquilo que se gosta. Ultimamente tenho focado meu aprendizado no framework Django.
